# Build Inputs

Place generated or captured input files for image generation in this directory.

Common files:

- `config_space_4k.bin`
- `config_space_parsed.json`
- `bar_info.json`
- `nvme_bar0.bin`
- `nvme_registers_parsed.json`
- `identify_controller.bin`
- `identify_namespace.bin`

These files are consumed by the scripts in `pipeline/`.
